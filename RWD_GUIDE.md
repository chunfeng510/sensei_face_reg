- **手機 (xs)**: < 600px

### 3. 行動裝置特殊優化

#### 防止不必要的縮放
```css
/* 輸入框字體最小 16px，防止 iOS Safari 自動放大 */
inputProps={{ style: { fontSize: '16px' } }}
```

#### 觸控友善
- ✅ 所有按鈕最小高度 44px（Apple 觸控標準）
- ✅ 使用 `touch-action: manipulation` 防止雙擊縮放
- ✅ 增加按鈕內距提高可點擊性

#### Viewport 設定
```html
<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=5, user-scalable=yes, viewport-fit=cover" />
```

#### 安全區域支援（iPhone X 系列）
```css
padding-bottom: env(safe-area-inset-bottom);
```

#### PWA 支援
- ✅ Apple 裝置網頁應用程式支援
- ✅ 主題顏色設定
- ✅ 禁用電話號碼自動偵測

### 4. 效能優化

- ✅ 圖片自動縮放（`max-width: 100%`）
- ✅ 使用 `-webkit-overflow-scrolling: touch` 提升滾動流暢度
- ✅ 防止水平滾動（`overflow-x: hidden`）

2. 點擊 Device Toolbar 圖示（或按 Ctrl+Shift+M）
3. 選擇不同裝置預設值測試：

## 💡 使用建議

### 最佳體驗
- 建議在 **直向模式** 下使用
- 網路連線穩定時上傳照片
- 照片解析度建議 1080p 或以下

### 省流量模式
- 避免上傳過大的照片
- 建議先壓縮照片再上傳

## 🎨 Material-UI 響應式工具

系統使用 Material-UI 的 `sx` prop 實現響應式：

```javascript
sx={{
  fontSize: { xs: '0.875rem', sm: '1rem', md: '1.125rem' },
  p: { xs: 2, sm: 3, md: 4 },
  display: { xs: 'block', sm: 'flex' }
}}
```

### 斷點對應
- `xs`: 0px+（手機）
- `sm`: 600px+（大手機/小平板）
- `md`: 900px+（平板）
- `lg`: 1200px+（小桌機）
- `xl`: 1536px+（大桌機）

## 🐛 已知問題與解決方案

### iOS Safari 輸入框放大
**解決**: 所有輸入框 `fontSize: '16px'`

### Android 鍵盤遮擋輸入框
**解決**: 對話框使用 `maxHeight: '90vh'`

### 圖片載入閃爍
**解決**: 加入 loading 狀態和過渡效果
