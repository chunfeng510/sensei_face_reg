# Sensei Face Recognition - Frontend Setup

## 安裝依賴

```bash
cd frontend
npm install
```

## 啟動開發伺服器

```bash
npm start
```

應用程式會在 http://localhost:3000 啟動

## 專案結構

```
frontend/
├── public/
│   └── index.html
├── src/
│   ├── components/      # React 元件
│   ├── services/        # API 服務
│   ├── App.js          # 主應用程式
│   └── index.js        # 入口檔案
├── package.json
└── README.md
```

## 開發指令

- `npm start` - 啟動開發伺服器
- `npm build` - 建置生產版本
- `npm test` - 執行測試
